Mini E-Commerce Web Application – Client Side (Deliverable 1)


My Development Journey
----------------------
Hey! I'm really excited to share my work on "MiniShop", my submission for Deliverable 1 of the CMP7246 Web Application Development module. I built this entirely from scratch over the past few weeks to demonstrate everything I've learned so far about frontend development.

The biggest challenge I faced was definitely getting the vanilla JavaScript shopping cart to work smoothly with LocalStorage. It took me quite a bit of trial and error to handle state updates without a framework, but I'm incredibly proud of how robust the cart logic turned out. Styling the site without Bootstrap or Tailwind was also a great learning experience—I relied heavily on CSS Grid and Flexbox, and I think the responsive design looks pretty solid.

Please note that as per the assignment instructions for Deliverable 1, there is no backend or live database attached yet. The login and signup forms are purely for UI and client-side validation demonstration right now. I spent extra time making sure the regex validation on those forms provides helpful error messages!

What I Built
------------
1. index.html    – The main landing page showcasing featured mock sections.
2. products.html – A dynamic product catalog where items can be filtered by category.
3. cart.html     – A fully working cart relying on LocalStorage to persist items.
4. login.html    – A styled login interface featuring regex-based client-side validation.
5. register.html – A signup form with password confirmation checks.
6. about.html    – A personal page explaining my project's background.

How to Run My Project
---------------------
1. Extract the zip file containing my code.
2. Open the project folder on your computer.
3. Double-click the `index.html` file to open it in your preferred browser.
4. Feel free to navigate around and try adding items to the cart across different pages!
