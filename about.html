About MiniShop

Our Story

Hi there! Welcome to the story behind MiniShop. I built this entire platform from the ground up as my university project for the CMP7246 Web Application Development module. It's been quite the journey creating a fully functional e-commerce site from scratch.

One of the biggest hurdles was getting the vanilla JavaScript shopping cart to work perfectly with local storage, and ensuring the responsive design looked great on both mobile and desktop screens. I've learned so much about structuring complex web layouts and managing state without relying on heavy frameworks.

Our Technologies

For this project, I decided to stick to the core web foundations. I used semantic HTML5 to ensure the content is accessible and well-structured. For styling, I wrote custom CSS3 from scratch, relying heavily on modern layout techniques like Flexbox and CSS Grid to make sure the interface adapts smoothly to any device size. When it came to interactivity, I chose to use vanilla JavaScript rather than reaching for a library. This gave me a profound understanding of DOM manipulation, handling dynamic cart state, and implementing robust form validations entirely on my own.
